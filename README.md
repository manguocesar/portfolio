# Overview
Portfolio Next app to showcase freelance projects & web development skills

## To run the app

1.  **Install the dependencies.**
    ```shell
    pnpm
    ```

2.  **Run the app locally.**
    ```shell
    pnpm run dev
    ```
Open the browser at http://localhost:3000/

## Additional
Hosted with [Vercel](https://vercel.com/)
